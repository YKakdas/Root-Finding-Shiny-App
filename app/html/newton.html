Newton-Raphson

The Newton-Raphson method (also known as Newton's method) approximates the root of a real-valued function f(x) = 0. It uses the idea that a continuout and differentiable function can be approximated by a straight line tangent to it.

At each iteration (also called Newton Step), new approximation is calculated by using following equation,

xn+1 = xn - f(xn) / f'(xn)

Running Instructions

Provide an input function and an initial value by following the rules given below and hit the Calculate button. The program will output a table with results and a plot if algorithm converges. You may enlarge and download the plot by clicking on it. You may click on the Reset button for clearing the input fields and outputs. Although the Newton's Method requires a derivative of the function, the program handles the derivative by itself, therefore only providing the input function will be sufficient.

Tips

  • Multiplication should be entered by using "asterisk(*)" such as 3 * x (Do not type 3x)

  • Any trigonometric function should be used such as: cos(x), sin(x), atan(x),...

  • Power of Euler Number(e) should be entered using "exp" such as exp(x)

  • Any other exponentials can be typed using "^" such as x^-5

  • An initial value(x0) should be given for the algorithm. Please only enter a numeric value.

  • Example query: Function: log(x) - exp(-x)  Initial Value: 2
