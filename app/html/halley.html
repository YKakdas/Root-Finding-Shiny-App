Halley's Root Finding

The Halley's method (also known as tangent hyperbolas method) approximates the root of a real-valued function f(x) = 0. The basic idea is to find an approximation of the function that both matches first and second derivatives at the current point and crosses the x-axis.

At each iteration, new approximation is calculated by using following equation,

xn+1 = xn - 2f(xn) f'(xn) / (2f'(xn)^2 - f(xn)f''(xn))

Running Instructions

Provide an input function and an initial value by following the rules given below and hit the Calculate button. The program will output a table with results and a plot if algorithm converges. You may enlarge and download the plot by clicking on it. You may click on the Reset button for clearing the input fields and outputs. Although the Halley's Method requires a first and second derivative of the function, the program handles the derivative by itself, therefore only providing the input function will be sufficient.

Tips

  • Multiplication should be entered by using "asterisk(*)" such as 3 * x (Do not type 3x)

  • Any trigonometric function should be used such as: cos(x), sin(x), atan(x),...

  • Power of Euler Number(e) should be entered using "exp" such as exp(x)

  • Any other exponentials can be typed using "^" such as x^-5

  • An initial value(x0) should be given for the algorithm. Please only enter a numeric value.

  • Example query: Function: x^3 -x -1 Initial Value: 1.5
